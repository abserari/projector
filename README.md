# projector

face detect and IOT data display

## TechStack 
- antdesign pro
- face api
- g2 for plot
- emitter for pubsub
- aliyun sdk 

## Face Usage
- use own aliyun apikey

- run emitter and mysql in face/mysql&emitter

- run face api model file host in weights

- run frontend 

- run backend 

- run fileserver to have image display.
